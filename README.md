# Rock Paper Scissors 🎮

This is my first Python project!   

---

## How to Play
- Run the program in Python.
- Type:
  - `r` for Rock 🪨  
  - `p` for Paper 📄  
  - `s` for Scissors ✂️  
- The computer will pick randomly and the result will be shown.

---

## Example
Enter your choice: r
You chose Rock
Computer chose Scissors
You win! 🎉
